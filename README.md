# schuylkill-stem-command
Schuylkill STEM Command Center (IU29)
Mission: Providing a world-class makerspace experience for every student in Schuylkill County ("The Skook").
This repository contains the STEM Command Center, a centralized platform designed for the IU29 STEM & District Support office. It serves three primary functions:
	1	Public Showcase: A high-impact "Wall of Fame" for county-wide STEM success.
	2	Makerspace Pitch Portal: A direct line for teachers and students to request equipment and funding.
	3	Community Bridge: A recruitment hub for retired machinists, engineers, and local manufacturers to volunteer as mentors.
🚀 Deployment (How to go Live)
The easiest way to get this in front of the community is using GitHub Pages or Vercel:
	1	Create a Repository: Name it schuylkill-stem-command.
	2	Upload the Code: Save the STEM_Command.jsx logic into a standard React structure (e.g., created via vite).
	3	Host for Free: * Enable GitHub Pages in the repository settings.
	◦	Result: You'll have a live URL (e.g., iu29-stem.github.io) to show at board meetings.
🛠 Features for District Support
For Teachers & Students
	•	The Pitch: No more messy emails. All makerspace requests are routed through the "Project Pitch" form directly to the Supervisor's dashboard.
	•	Visibility: Showcase projects from all 12 districts (Pottsville, Blue Mountain, North Schuylkill, etc.) on one screen.
For Local Industry (Mentors)
	•	The "Skook" Connection: A specialized portal for retired pros to register their skills (Welding, CNC, Coding).
	•	Manufacturer Sync: Allows local plants (ALCOA, Hydro, etc.) to see where their expertise is needed most.
📈 Tech Stack
	•	Framework: React
	•	Styling: Tailwind CSS (Custom "Black-weighted" typography & Glassmorphism)
	•	Icons: Lucide-React (High-performance SVG)
	•	Architecture: Single-file component for rapid deployment and easy maintenance.
